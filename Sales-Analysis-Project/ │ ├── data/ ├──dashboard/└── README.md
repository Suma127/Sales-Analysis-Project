# Sales-Analysis-Project
# Sales & Business Performance Analysis
# Project Overview:
This project analyzes sales data to identify key business insights such as top-performing regions, products, and discount impact.
# Objectives:
- Analyze sales performance
- Identify top regions and products
- Understand discount trends
- Build interactive dashboard
# Tools Used:
- Power BI
- Excel
# Project Workflow:
- Data Cleaning using Power BI
- Data Analysis (aggregation, trends)
- Dashboard creation in Power BI
# Key Insights:
- Revenue growth driven primarily by South (+36%) and North (+26%).
- West region shows stagnation (0% YoY) despite stable revenue base.
- Tablet category underperforming relative to other products.
- Growth appears constrained by declining order volume and AOV.
# Project Structure:
- data/ → raw and cleaned data
- Power BI/ → analysis and create dashboard
# Outcome:
This project demonstrates end-to-end data analysis from data cleaning to visualization and business insights.
